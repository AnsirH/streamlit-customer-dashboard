# Streamlit Customer Dashboard

## 요구사항

- Python 3.9 이상, 3.12 미만
- pip 24.0 이상

## 설치 방법

1. 가상환경 생성 및 활성화:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

2. 패키지 설치:
```bash
pip install -r requirements.txt
```

3. 앱 실행:
```bash
streamlit run app.py
```

## 주요 기능

- 고객 이탈 예측
- 데이터 시각화
- 통계 분석
